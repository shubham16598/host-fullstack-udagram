The Infrastruture uses AWS Cloud for hosting purpose:

1) Elastic Beanstalk: It is used to host the backend API built on nodejs.

2) S3 Bucket: To host the frontend static files, The bucket policy is changes to make it accessible from the frontend.

3) Amazon RDS: RDS Server is used to store the database of the application
